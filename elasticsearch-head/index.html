elasticsearch-head

What is this?

elasticserach-head is a web front end for browsing and interacting with an Elastic Search cluster.

Gettings Started

  1. git clone git://github.com/mobz/elasticsearch-head.git
  2. Open index.html in a browser. A modern browser is required to use es-head
  3. By default es-head attempts to connect to a node at http://localhost:9200/. Enter a different node address in the connect box and click 'Connect'

General Utility

es-head has a three major operations.

  • A couple of search interfaces that allow you to query the cluster a retrieve results in raw json or tabular format
  • Several quick access tabs that show the status of the cluster
  • An input section that allows arbitrary call to the RESTful API to be made.
